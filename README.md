# Backup

备份文件生成在Download/CC文件夹下面

### Permission
* android.permission.READ_SMS
* android.permission.READ_CALL_LOG
* android.permission.WRITE_CALL_LOG
* android.permission.WRITE_EXTERNAL_STORAGE
* android.permission.READ_EXTERNAL_STORAGE
